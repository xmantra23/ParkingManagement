How to compile:


1. Compile MavParkTest.java file in cmd after navigating to the correct directory
   (javac MavParkTest.java)

After successful compilation 

2. run the following command in cmd ( java MavParkTest input.txt output.txt)
   Note. input.txt is the input file provided in the folder.
         output.txt can be anything. it is the user defined output file.
